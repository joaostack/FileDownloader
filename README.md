# FileDownload
Experimental Project | Dotnet File Downloader, with a simple & beautiful interface

```
$ dotnet restore; dotnet run --project src -- download -h
  _____   _   _          ____                               _                       _

 |_|     |_| |_|  \___| |____/   \___/    \_/\_/   |_| |_| |_|  \___/   \__,_|  \__,_|  \___| |_|

USAGE:
    FileDownloader1.dll download [OPTIONS]

OPTIONS:
    -h, --help      Prints help information
    -o, --output    The output of downloaded file
    -u, --url       The URL of the file to download
```

## Dependencies
- [.NET 10](https://dotnet.microsoft.com/en-us/download)
- [Spectre.Console / Spectre.Console.Cli](https://spectreconsole.net/)
